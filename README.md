# RAG Document Assistant

An AI-powered document assistant that lets you upload 
multiple PDFs and ask questions across all of them.

## Features
- Multi-PDF upload and processing
- Conversational memory
- Source citations with page numbers
- Built with LangChain, Groq, and Streamlit

## Tech Stack
- Python
- LangChain
- Groq API (LLaMA 3.3 70B)
- ChromaDB
- Streamlit
- HuggingFace Embeddings
